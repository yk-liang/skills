# 评分体系（按票型分别打分）

总分 100。**红线复核优先于分数**。任何 red-flag 命中 → 直接进回避池，不论分数多高。

不同票型用**不同**的评分维度权重——情绪龙头不在乎财报，中线慢牛不在乎涨停板。

---

## 通用 5 维度（满分 100）

| 维度 | 权重 | 含义 |
|---|---:|---|
| 1. 价格位置与量价 | 25 | 趋势、5/20/60 日涨幅、是否加速、量价配合 |
| 2. 财报质量（时效校正） | 25 | 扣非/经营现金流为主；披露超 60 天衰减 ×0.7、超 120 天 ×0.5 |
| 3. 公告催化 | 20 | 近 30 日可核验公告（订单、回购、政策、业绩预告等） |
| 4. 板块位置 | 15 | 主线 / 分支 / 补涨 / 跟风 |
| 5. 次日触发 | 15 | 前日尾盘资金、集合竞价预期、盘前消息、大盘缺口、明日关键技术位 |

> 2026-06 改造：原"账户适配度 20 分"维度删除（它是事后调节项不是"票好不好"），改为本文末「账户适配 — 仓位乘数 + 黑名单」机制。腾出权重给新增的"次日触发"维度 — 解决"评分高但今天买点已过"的高频问题。

---

## 1. 价格位置与量价（25 分）

| 区间 | 含义 |
|---|---|
| 20–25 | 趋势改善但未透支；20 / 60 日结构支持低风险介入；量能健康（量价齐升 + 缩量回踩） |
| 14–19 | 趋势向上但部分透支；可等回踩 |
| 8–13 | 趋势弱、震荡、远离支撑 |
| 0–7 | 已加速暴拉、连续涨停、高位高换手或耗尽 |

**追高阈值（硬封顶）**：
- 5D > 20% → 封顶 13 分
- 5D > 30% → 封顶 6 分
- 20D > 35% → 封顶 10 分
- 60D > 60% → 封顶 6 分

**量价规则**：
- 缩量上涨 → −3 分（"缩量上涨不追高"）
- 高位放量滞涨 → −5 分（典型出货信号）
- 高位低换手回调 → +2 分（可拿）
- **低位高换手** → +2 分（可挖掘）
- **低位低换手** → −3 分（不做）
- **高位高换手** → −5 分（赶紧跑）

---

## 2. 财报质量（25 分，含时效校正）

用最新季报 / 年报 / 业绩预告。**优先看扣非净利润和经营现金流**，比表观净利润可靠。

**时效衰减**（A 股财报披露期外用旧数据，必须打折）：
- 财报披露日距今 ≤ 60 天 → 评分 × 1.0
- 距今 60–120 天 → 评分 × 0.7（公告披露期之间，数据老化）
- 距今 > 120 天 → 评分 × 0.5（已过期，必须降权）
- 业绩预告 / 快报披露后 → 优先用预告/快报（视为最新，不打折）

**最新季度要求**（中线慢牛票型硬约束）：
- 6 月跑：必须有 2026Q1（4 月披露）；若只有 2025 年报 → 评分上限 60
- 9 月跑：必须有 2026 半年报（8 月披露）；若只有 2026Q1 → 评分上限 60
- 11 月跑：必须有 2026Q3（10 月披露）
- 短线票型（短线龙头/老龙反抽/护盘衍生）无此硬约束 — 财报权重本身就低

| 区间 | 含义 |
|---|---|
| 21–25 | 营收 + 归母 + 扣非同比双正；毛利率稳定/向上；经营现金流为正且与利润匹配 |
| 16–20 | 主要指标可接受，但某一维度偏弱 |
| 10–15 | 增长不稳；扣非或现金流弱；毛利率有压力 |
| 0–9 | 亏损 / 扣非大幅恶化 / 应收账款异常 / 现金流与利润严重背离 |

**警告**：
- 利润高但扣非低 → 关注非经常性损益（政府补贴、卖资产）
- 利润高但经营现金流为负 → 应收账款堆积，−5 分

---

## 3. 公告催化（20 分）

只用可核验公告。**不接受任何"市场传闻""听说"**。

| 区间 | 含义 |
|---|---|
| 16–20 | 近 30 日有重大订单（>5% 年营收）/ 注销式回购 / 业绩预告大幅增长 / 政策直接受益 / 实质资产注入 |
| 11–15 | 温和催化或已部分price in |
| 6–10 | 无明确催化但也无负面 |
| 0–5 | 减持公告、问询函、大宗诉讼、集中质押、异动公告无支撑 |

**Red flag**（命中即 0 分 + 进回避池）：
- 立案调查、警示函、监管函
- 高比例质押 + 平仓预警
- 业绩预告大幅下滑
- "无应披露而未披露事项"声明
- 异常波动公告 + 无基本面支撑

**利好的 sanity check**：
- 中标合同看金额 / 年营收占比，<5% 不算硬催化
- 回购看是否注销式 + 总股本占比 >1% 才有意义
- 股权激励看行权价相对现价的折价

---

## 4. 板块位置（15 分）

| 区间 | 含义 |
|---|---|
| 13–15 | 当前主线 + 板块有宽度（多只票同涨）+ 该股是核心或可信分支 |
| 9–12 | 板块有动能 + 该股是合理分支或补涨 |
| 5–8 | 板块零散或纯炒作 |
| 0–4 | 纯题材 / 已过主升 / 跟风末端 |

**关键提示**：
- "**不要去买那些没有追随板块领涨股的股票**"——板块都在涨它纹丝不动 → 弱势 → 不要幻想后涨
- 大作手原则："和板块趋势相悖是不明智的"

---

## 5. 次日触发（15 分）

**不评"票本身好不好"**（前 4 维已覆盖），评"明日开盘后这只票什么时候能买、用什么价位买"。这一维是"评分高但买点已过"的过滤器 — 评分高但触发条件模糊的票降级。

| 区间 | 含义 |
|---|---|
| 12–15 | 触发条件明确：集合竞价高开 0–2% 即介入 / 平开等盘中冲过 X 价位 / 明确回踩 X 日线低吸；前日尾盘资金净流入；明日无大盘缺口压力 |
| 8–11 | 触发条件清晰但需要等 1–2 个盘中确认信号 |
| 4–7 | 触发条件模糊：要"看情况""等机会"；前日尾盘动能弱 |
| 0–3 | 无明确触发：今日买点已过、明日只能追高 / 等大幅回踩 |

**评分维度**：
1. **集合竞价预期**：根据昨日封板情况 + 板块情绪 + 大盘缺口判断高开/平开/低开概率
2. **盘前消息**：夜里美股 / 行业新闻 / 巨潮公告是否影响该股
3. **前日尾盘资金**：最后 30 分钟主力净流入（用 `quote.sh` 的 today_main_inflow，如有）
4. **明日关键技术位**：今日收盘价距 5/10/20 日线、前低、关键平台的距离
5. **大盘缺口**：今日上证收盘是否留有跳空缺口，明日是否需要回补

**反例**：评分 80 分但只能写"等回踩 5 日线，回踩到 36 介入"= 次日触发维度 ≤ 7 分，整体加权后降到 70 以下。

## 账户适配 — 仓位乘数 + 黑名单（替代原维度 5）

账户因素**不再作为加分维度**（这是事后调节项，不是"票本身好不好"），改为以下机制：

**仓位上限乘数**（票评分通过后，决定建仓量时乘进去）：
- 单日振幅 > 7% 长期 → 仓位上限 × 0.5
- 流通市值 30–50 亿 → 仓位上限 × 0.8
- 流通市值 < 30 亿 → 仓位上限 × 0.5（小盘股庄家影响过大）

**直接黑名单**（不入选）：
- 流通市值 < 15 亿（极小盘 + 流动性差）
- 5 日均成交额 < 5000 万（盘子薄、跑不掉）
- 5 日波动率（日振幅平均）> 9% 且无明确催化（妖股）

**板块仓位约束**：单板块合计 > 25% → 当晚不再从该板块新增候选

## 评级结果

| 总分 | 处置 |
|---:|---|
| 80–100 | 候选池。仍需源头核验、纪律红线、仓位控制 |
| 70–79 | 观察池。等更好价格 / 催化确认 |
| 60–69 | 低优先级。仅与持仓相关时提及 |
| < 60 | **不入选** |

## 票型 → 维度权重调整

不同票型用不同权重（在通用基础上**加权**）：

（2026-06 票型从 7 → 4，权重表对应缩到 4 行；账户适配维度将在评分改造中迁移为乘数封顶，本表保留过渡）

| 票型 | 价格位置 | 财报 | 公告 | 板块 | 账户适配 |
|---|---:|---:|---:|---:|---:|
| 中线慢牛 | ×1.0 | ×1.5 | ×1.0 | ×1.0 | ×1.2 |
| 短线龙头 | ×1.5 | ×0.5 | ×1.2 | ×1.5 | ×1.0 |
| 老龙反抽 | ×1.3 | ×0.6 | ×1.0 | ×1.2 | ×1.0 |
| 护盘衍生 | ×0.8 | ×1.2 | ×0.8 | ×1.5 | ×1.5 |

### 加权归一化算法（明确公式）

1. **通用打分**：对每维度按上面 5 个章节的区间表给原始分 raw_i（满分依各维度权重，即 25/25/20/15/15）
2. **维度归一化**：normalized_i = raw_i / max_i  （得到 [0, 1] 区间）
3. **按票型加权求和**：weighted_sum = Σ(normalized_i × w_i)  （w_i 取自票型权重表）
4. **按权重和归一**：final_score = weighted_sum / Σ(w_i) × 100

**示例**（短线龙头某候选）：
- 通用打分：价格 18/25、财报 12/25、公告 14/20、板块 12/15、次日触发 9/15
- 归一化：0.72 / 0.48 / 0.70 / 0.80 / 0.60
- 短线龙头权重：1.5 / 0.5 / 1.2 / 1.5 / 1.0
- 加权求和：0.72×1.5 + 0.48×0.5 + 0.70×1.2 + 0.80×1.5 + 0.60×1.0 = 1.08 + 0.24 + 0.84 + 1.20 + 0.60 = 3.96
- Σ(w_i) = 5.7
- final_score = 3.96 / 5.7 × 100 = **69.5**

**报告里写**：票型 + 最终分 + 关键扣分项（如"次日触发 9/15，需明日开盘 < 36 才入场"），让用户看到分数构成不只是一个数字。

## Red-flag overrides（命中即进回避池）

无论分数多高，以下任一命中**直接降级为"回避"**：

1. **选股池外**（**仅选股层** — 评估层放开）：选股建议仅在主板（600/601/603/605/000/001/002/003）；创业板 300/301/302、科创板 688/689、北交所 8/4/920、B 股 900/200 **可评估作为板块风向标 / 龙头识别参考**，但不进个股候选池
2. **ST/*ST、退市整理、上市不足 60 个交易日的次新**
3. **大股东减持公告**（最近 30 日）+ 减持比例 ≥ 1% — 注意区分：`shareholder_reduction=true`（新计划/正在减持）才是红旗；`shareholder_reduction_completed=true`（减持实施完毕/终止/承诺不减持）是利空落地，**不进回避池**，反而可视为"抛压释放"中性偏好信号
4. **集中质押、平仓预警、商誉问询**
5. **异常波动公告 + 无基本面支撑**
6. **连续一字板**（散户没机会买进，不要幻想"我能挂到"；与 #9 追连扳重叠但单列因散户根本买不到这个独立原因）
7. **财务造假调查、监管处罚、重大未结诉讼**
8. **缩量启动几天后才放量**（用户明确禁止 — 新能泰山案例）
9. **追连扳**（软化版：见 case-library 反例 — 不是"≥3 涨停即回避"。仅在"自己没在底部就上车 + 大盘转弱"时回避；高位接力前提下可由"短线龙头子型 A"分类做，但仓位上限 ×0.5）
10. **大盘 C 级 + 短线票**（违反保命线）
11. **退潮期 + 首板/反包/新题材**（违反退潮三不做）

**降级（非 override，标"低置信度"在报告显式提醒）**：
- 数据三家都拿不到 → 标"低置信度"而非"回避"（endpoint 故障的好票不应被误杀）；报告写"数据缺口 X，候选保留但置信度低"
- 单一数据源指标异常但其他源正常 → 标"数据冲突"
