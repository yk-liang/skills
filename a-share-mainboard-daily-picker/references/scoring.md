# 评分体系（按票型分别打分）

总分 100。**红线复核优先于分数**。任何 red-flag 命中 → 直接进回避池，不论分数多高。

不同票型用**不同**的评分维度权重——情绪龙头不在乎财报，中线慢牛不在乎涨停板。

---

## 通用 5 维度（满分 100）

| 维度 | 权重 | 含义 |
|---|---:|---|
| 1. 价格位置 | 20 | 趋势、5/20/60 日涨幅、是否加速、量价配合 |
| 2. 财报质量 | 25 | 营收/归母/扣非/经营现金流/毛利率 |
| 3. 公告催化 | 20 | 近 30 日可核验公告（订单、回购、政策、业绩预告等） |
| 4. 板块位置 | 15 | 主线 / 分支 / 补涨 / 跟风 |
| 5. 账户适配度 | 20 | 流动性、波动、止损清晰度、加仓空间 |

---

## 1. 价格位置（20 分）

| 区间 | 含义 |
|---|---|
| 16–20 | 趋势改善但未透支；20 / 60 日结构支持低风险介入；量能健康（量价齐升 + 缩量回踩） |
| 11–15 | 趋势向上但部分透支；可等回踩 |
| 6–10 | 趋势弱、震荡、远离支撑 |
| 0–5 | 已加速暴拉、连续涨停、高位高换手或耗尽 |

**追高阈值（硬封顶）**：
- 5D > 20% → 封顶 10 分
- 5D > 30% → 封顶 5 分
- 20D > 35% → 封顶 8 分
- 60D > 60% → 封顶 5 分

**量价规则**：
- 缩量上涨 → −3 分（"缩量上涨不追高"）
- 高位放量滞涨 → −5 分（典型出货信号）
- 高位低换手回调 → +2 分（可拿）
- **低位高换手** → +2 分（可挖掘）
- **低位低换手** → −3 分（不做）
- **高位高换手** → −5 分（赶紧跑）

---

## 2. 财报质量（25 分）

用最新季报 / 年报 / 业绩预告。**优先看扣非净利润和经营现金流**，比表观净利润可靠。

| 区间 | 含义 |
|---|---|
| 21–25 | 营收 + 归母 + 扣非同比双正；毛利率稳定/向上；经营现金流为正且与利润匹配 |
| 16–20 | 主要指标可接受，但某一维度偏弱 |
| 10–15 | 增长不稳；扣非或现金流弱；毛利率有压力 |
| 0–9 | 亏损 / 扣非大幅恶化 / 应收账款异常 / 现金流与利润严重背离 |

**警告**：
- 利润高但扣非低 → 关注非经常性损益（政府补贴、卖资产）
- 利润高但经营现金流为负 → 应收账款堆积，−5 分

---

## 3. 公告催化（20 分）

只用可核验公告。**不接受任何"市场传闻""听说"**。

| 区间 | 含义 |
|---|---|
| 16–20 | 近 30 日有重大订单（>5% 年营收）/ 注销式回购 / 业绩预告大幅增长 / 政策直接受益 / 实质资产注入 |
| 11–15 | 温和催化或已部分price in |
| 6–10 | 无明确催化但也无负面 |
| 0–5 | 减持公告、问询函、大宗诉讼、集中质押、异动公告无支撑 |

**Red flag**（命中即 0 分 + 进回避池）：
- 立案调查、警示函、监管函
- 高比例质押 + 平仓预警
- 业绩预告大幅下滑
- "无应披露而未披露事项"声明
- 异常波动公告 + 无基本面支撑

**利好的 sanity check**：
- 中标合同看金额 / 年营收占比，<5% 不算硬催化
- 回购看是否注销式 + 总股本占比 >1% 才有意义
- 股权激励看行权价相对现价的折价

---

## 4. 板块位置（15 分）

| 区间 | 含义 |
|---|---|
| 13–15 | 当前主线 + 板块有宽度（多只票同涨）+ 该股是核心或可信分支 |
| 9–12 | 板块有动能 + 该股是合理分支或补涨 |
| 5–8 | 板块零散或纯炒作 |
| 0–4 | 纯题材 / 已过主升 / 跟风末端 |

**关键提示**：
- "**不要去买那些没有追随板块领涨股的股票**"——板块都在涨它纹丝不动 → 弱势 → 不要幻想后涨
- 大作手原则："和板块趋势相悖是不明智的"

---

## 5. 账户适配度（20 分）

针对 ≈ 13.5 万账户 + 当前回撤背景。

| 区间 | 含义 |
|---|---|
| 16–20 | 流动性好、波动可控、可分批建仓、止损清晰、不需要重仓 |
| 11–15 | 候选 OK 但波动 / 流动性需要降低仓位 |
| 6–10 | 高 beta / 事件驱动，只能小仓位试错 |
| 0–5 | 需要重仓博、跳空风险、流动性差、可能加重回撤 |

**自动降级触发**：
- 单日振幅 > 7% 长期 → 降到 ≤ 10 分（不适合修复回撤的账户）
- 流通市值 < 30 亿 → 降到 ≤ 12 分（小盘股庄家影响过大）
- 单票若入选会导致单板块仓位 > 25% → 强制降级

---

## 评级结果

| 总分 | 处置 |
|---:|---|
| 80–100 | 候选池。仍需源头核验、纪律红线、仓位控制 |
| 70–79 | 观察池。等更好价格 / 催化确认 |
| 60–69 | 低优先级。仅与持仓相关时提及 |
| < 60 | **不入选** |

## 票型 → 维度权重调整

不同票型用不同权重（在通用基础上**加权**）：

| 票型 | 价格位置 | 财报 | 公告 | 板块 | 账户适配 |
|---|---:|---:|---:|---:|---:|
| 情绪龙头 | ×1.5 | ×0.3 | ×1.2 | ×1.5 | ×1.0 |
| 老龙反抽 | ×1.3 | ×0.5 | ×1.0 | ×1.2 | ×1.0 |
| 错杀反弹 | ×1.5 | ×0.7 | ×1.5 | ×0.8 | ×1.2 |
| 护盘衍生 | ×0.8 | ×1.2 | ×0.8 | ×1.5 | ×1.5 |
| 中线慢牛 | ×1.0 | ×1.5 | ×1.0 | ×1.0 | ×1.2 |
| 均线启动 | ×1.5 | ×0.8 | ×1.0 | ×1.5 | ×1.0 |
| 逆市不跌 | ×1.3 | ×1.2 | ×1.2 | ×1.0 | ×1.0 |

实际打分先打通用 5 维 → 按票型乘权重 → 归一化回 100 分制。报告里写票型并显示加权后的分。

## Red-flag overrides（命中即进回避池）

无论分数多高，以下任一命中**直接降级为"回避"**：

1. **股票池外**：科创板 688/689、创业板 300/301/302、北交所 8/4/920、B 股 900/200（除非用户明确放开）
2. **ST/*ST、退市整理、上市不足 60 个交易日的次新**
3. **大股东减持公告**（最近 30 日）+ 减持比例 ≥ 1%
4. **集中质押、平仓预警、商誉问询**
5. **异常波动公告 + 无基本面支撑**
6. **连续一字板**（散户没机会买进，不要幻想"我能挂到")
7. **财务造假调查、监管处罚、重大未结诉讼**
8. **数据无法核验**（巨潮、东财、同花顺三家都拿不到，不要硬选）
9. **缩量启动几天后才放量**（用户明确禁止 — 新能泰山案例）
10. **追连扳**（用户明确禁止 — 东方明珠案例）
11. **大盘 C 级 + 短线票**（违反保命线）
12. **退潮期 + 首板/反包/新题材**（违反退潮三不做）
