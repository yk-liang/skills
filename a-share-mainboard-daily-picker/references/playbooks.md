# 操作手册（用户经验沉淀的可执行版本）

这份文档把用户 Notion 里的复盘经验拆成可直接执行的判定。每条规则都标注**为什么**——不只为了照办，也为了 agent 在边界场景能合理判断。

---

## 1. 大盘判级（黄白线五眼框架）

### 数据来源
东方财富上证分时图 + 资金流。打开 `https://quote.eastmoney.com/zs000001.html` 看分时图的两条线：

- **白线** = 加权指数（受银行/保险/石油/白酒等权重股影响大）
- **黄线** = 平均股价线（不加权，反映多数个股的真实赚钱效应）

### 五眼检查
1. **黄白线相对位置**：黄≥白（情绪好）、黄<白（赚指数不赚钱）。
2. **10:30 前结构**：高开后横住 = 资金不急跑；拉一下就被砸 = 护指数出货。
3. **成交量**：放量但黄线不涨 = 权重在出货 / 缩量但黄线稳 = 情绪未崩。
4. **资金（BBD/主力净流入）**：净流入 = 真涨；持续负流入但指数稳 = 国家队/权重在扛。
5. **13:30 后是否有回流**：午后量没跟上 = 直接放弃，午后量起来 = 可加仓。

### 定级
| 级别 | 黄白线 | 量 | 资金 | 仓位 | 短线策略 | 中线策略 |
|---|---|---|---|---|---|---|
| **A** | 黄 ≥ 白，共振向上 | 放量 | 净流入或回流 | 短 30–40% / 中 40–50% / 现 10–20% | 可追强、打板、吃情绪 | 可加仓、持股 |
| **B** | 白稳黄略弱 | 一般 | 小幅流出 | 短 10–20% / 中 40% / 现 40–50% | 只做最强分支 / 龙头 | 不加仓但可持有 |
| **C** | 黄明显弱于白 | 缩量或放量下跌 | 持续流出 | 短 0–5% / 中 ≤ 20% / 现 ≥ 70% | 基本不做 | 减仓、等右侧 |

**保命线**（必须执行，不准妥协）：
> **黄线连续 2 天弱于白线 + 量不放 → 所有短线仓位自动清零。**

由于本 skill 是夜间运行，用户已经知道当天黄白线全貌，所以判级看**今日**和**昨日**两天的形态。

## 2. 情绪温度 + 退潮期识别

### 情绪温度指标（每晚必抓）
- 涨停家数（剔除 ST）、跌停家数
- 连板高度（最高板 + 连板≥3 的票数）
- 昨日龙头今日表现（涨停 / 高开不封 / 低开破板）
- 首板晋级率：昨日首板今日二板的比例

### 退潮期判定（命中 ≥ 2 条 → 进入退潮模式）
1. 涨停家数连续下降
2. 昨日龙头**高开不封 / 冲高回落**
3. 昨强板块**无一字、无换手二板**
4. 黄线持续弱于白线
5. 首板晋级率 < 30%

### 退潮期纪律（铁律，不准违反）
- ❌ **不做首板**
- ❌ **不追反包**
- ❌ **不做新题材**
- 退潮期只做 3 类（按优先级，对应新 4 票型分类）：
  1. **老龙反抽子型 A**（前 5 板以上 + 跌 30–50% + 首次放量止跌）
  2. **护盘衍生 / 中特估对冲**（银行/石油/电力/中字头/中特估 + 低波动）
  3. **老龙反抽子型 B**（极端情绪错杀：无明显利空 + 1–2 天暴跌 15%+ + 尾盘有承接）
- 退潮期仓位公式：`涨停家数 × 1% = 当日最大短线仓位`
  - 例：涨停 30 家 → 短线 ≤ 30%；涨停 15 家 → 短线 ≤ 15%

### 情绪冰点的 5 个**提前**信号（命中 ≥ 2 → 立即降仓）
1. 龙头"给你走"了（低开能让你走、冲高不封不杀）—— **最危险的信号**，市场不抢了
2. 二板开始大量断板（首板还行但二板接不上）
3. 强股缩量阴跌（不暴跌但每天跌一点 + 量越来越小）
4. 跌停不多但涨停更少（看似没事，实际没人赚钱）
5. 同题材内卷（无唯一核心，互相卡位）

命中 ≥ 2 → 报告里**显式提醒用户**：
- 短线仓位直接减半
- 不再做二板以上
- 交易周期压缩到隔日

> **冰点 ≠ 抄底点。** 冰点 = 生存考验。真正的机会在冰点之后第一个"能连板的东西"出现时。

## 3. 票型分类与识别

候选股**必须明确归类**到下面**四种**之一（2026-06 简化：原 7 票型合并到 4 票型 + 1 观察池标签，13.5 万账户不养复杂分类）。不同票型用不同评分维度（详见 scoring.md）、不同仓位上限、不同止盈止损规则。

> **先板块再票型**：每个候选必须先回答"该股属于哪个板块、板块当前阶段是什么"。板块判定见 `references/sector-screening.md`。详细的板块阶段 → 票型映射表见本节末尾。
>
> **观察池标签**（如"逆市不跌"）不算独立票型 — 入选后归到合适的票型管理。

### 票型 1：中线慢牛（账户的"压舱石"）

**适用场景**：账户的主仓位，不绑定板块阶段，独立筛选。

**触发条件**（必须**全部**满足）：
1. 周线没走坏
2. 日线在 20 / 60 日线上
3. 回调缩量、上涨放量
4. 财报：营收 + 归母 + 扣非同比双正，经营现金流为正
5. 市场共识高（机构票，不纯炒作）
6. 日振幅 ≤ 3%（低波动）

**观察仓 → 主仓**升级条件（满足下列 6 选 5；2026-06 加第 6 条缠论 ③ 周线共振）：
1. 至少一个指数站回 60 日线 + 3 天不破
2. 黄白线出现至少 1 天共振
3. 回调缩量 + 突破放量（不是一日游）
4. 连续 3 天涨多跌少 + 赚钱效应 > 50%
5. 同一主线持续 5–7 天（非一日题材）
6. **周线 MACD 在 0 轴上方 + 周线 MA5/10/20 多头排列**（缠论"周线买点 > 日线"原则）
   - 验证：`./scripts/data/kline.sh <code> 60 1w | jq '{dif: .data.macd_last.dif, dea: .data.macd_last.dea, ma5: .data.ma5, ma10: .data.ma10, ma20: .data.ma20}'`
   - 通过条件：dif > 0 且 dea > 0 且 ma5 > ma10 > ma20
   - 作用：淘汰"日线漂亮但周线已是末段"的伪机会 — 这是中线慢牛防站岗的硬防线

**仓位节奏**：观察仓 20% → 突破确认加到 40% → 回踩不破加到 60%

**退仓条件**（任一触发）：
1. 跌破 60 日线
2. 放量长阴
3. 指数转 C 级 → 主仓退回观察仓 20%

**止盈位**：触发"三段式卖法"任一信号（强一致日 / 第一次放量震荡 / 跌破 5 日线），不设固定百分比目标。

---

### 票型 2：短线龙头（合并原"情绪龙头" + "均线启动"）

**适用场景**：机动仓主力。在主升 / 新启动板块内做"情绪龙头延续"或"均线金叉启动"。

**触发条件**（满足任一子型即可）：

- **子型 A — 情绪龙头型**：
  - 所在板块涨停家数第一
  - 分时第一次分歧能接住
  - 不看估值只看承接
  - **限定 A 级或强 B 级盘**，C 级直接跳过
  - **不打首板**（除非 A 级盘且确认是板内绝对龙头 — 涨停家数第一且板内只有它一只主板涨停 + 板块当日新启动）

- **子型 B — 三类买点型**（缠论 ① 简化版 — 2026-06 升级，原"均线启动"）：
  - **首选触发**：突破最近 30 日 K 线中枢上沿 + 回踩中枢上沿不破（容差 ≤ 3%）
  - 信号源：`./scripts/data/kline.sh <code> 60 1d | jq '.data.chanlun_levels[] | select(.type=="三类买点")'`
  - 中枢字段：`.data.zhongshu.upper` / `.data.zhongshu.lower` — 上沿是天然止损位
  - **备选触发**（若中枢字段 valid=false，即趋势段中无明显盘整）：5 日线上穿 10 日线 + 板块共振 + 等"成功穿上且走稳"（刚穿上去经常马上又下来）
  - 信号最强（两者共振）：三类买点 + 所有均线（5/10/20/30/60）多头排列 + 股价在均线上
  - 均线**低位缠绕**还好（向上概率大），**高位缠绕**是分歧（不做）
  - **止损**：跌破 `zhongshu.upper`（中枢上沿）即止损 — 比固定 3% 更符合缠论结构

**单票仓位**：≤ 8%

**止损**（取严者）：
- 结构止损：跌破当日 5 日线 / 当日前低 / 分时均线无承接
- 底线：单笔亏损 ≤ 3%

**止盈**：
- 三段式（强一致日 30% / 第一次放量震荡 30% / 跌破 5 日线全清）
- 子型 B 升级到多头排列后，跌破 10 日线再走

---

### 票型 3：老龙反抽（合并原"老龙反抽" + "极端情绪错杀" + "补涨板块错杀反弹"）

**适用场景**：退潮期 / 修复 / 补涨板块专用。三种子型来源不同但操作模式一致 — 都是"超跌或低位首次放量"。

**触发条件**（满足任一子型即可）：

- **子型 A — 个股老龙反抽**：
  - 前期 5 板以上
  - 跌幅 30–50%
  - 某天首次放量止跌
  - 黄线不再创新低
  - 适用：修复板块 / 退潮期

- **子型 B — 极端情绪错杀**：
  - 无明显利空
  - 1–2 天暴跌 15%+
  - 尾盘有承接
  - 不需要前期连板背景
  - 适用：退潮期 / 任意板块（不绑定板块阶段）

- **子型 C — 补涨板块温和跟随**（仅当主升板块已走 3 天 + 该板块今天才动 + 板内温和涨幅 5–7% 时启用）：
  - 板内尚未启动的低位主板股
  - 不追板内已涨停的票，只在补涨股内挑
  - 适用：补涨板块

**单票仓位**：
- 子型 A / B：≤ 10%
- 子型 C：≤ 5%

**操作周期**：只做隔日 / 两日

**止损**：跌破当日承接低点 / 进场点 −3% 取严者

**止盈**：反抽 5–10% 就走，不格局

---

### 票型 4：护盘衍生 / 中特估对冲

**适用场景**：C 级盘或退潮期的避风港。不绑定板块阶段，独立筛选。

**触发条件**：
- 板块属性：银行 / 石油 / 电力 / 中字头 / 中特估
- 日振幅 ≤ 3%（低波动）
- 当前不在加速段（5D 涨幅 < 8%）

**单票仓位**：≤ 15%

**止损**：跌破 20 日线 / 板块整体破位

**止盈**：当避风港用，不格局；大盘恢复 A 级 → 减仓换主升票

---

### 观察池标签：逆市不跌（不算独立票型）

发现后进观察池**不主动追**，等大盘企稳后第一个**放量大阳**跟进 — 介入时归"短线龙头"票型管理。

**触发条件**：
- 大盘调整时这只票横盘不跌
- 量能温和
- 无负面公告

**思路**：下轮黑马由此产生概率高（参考永鼎案例 — 3 月行情差时它能横盘不跌就是强）

---

### 票型 × 板块阶段映射表

从 Step 5 板块筛选出的阶段，决定从哪些板块内选什么票型：

| 板块阶段 | 允许的票型 | 仓位调节 |
|---|---|---|
| 主升 | 短线龙头（子型 A / B 均可） | A 级满仓上限 / B 级减半 |
| 新启动 | 短线龙头子型 A（仅 A 级）/ 子型 B（A / B 均可） | B 级仓位 ≤ 5% 试错 |
| 修复 | 老龙反抽（子型 A 优先） | 标准 |
| 补涨 | 老龙反抽子型 C | ≤ 5%（仓位减半） |
| 轮动 | 不选 | — |
| 退潮 | 老龙反抽子型 A / B / 护盘衍生 | 仓位公式 = 涨停家数 × 1% |

**中线慢牛 / 护盘衍生不绑定板块阶段，独立筛选。**

### 不允许选的（绝对禁区）
- 缩量启动几天后才放量介入的（新能泰山案例 — 一天涨停一天跌停，亏 20%）
- 连扳后追高（东方明珠案例 — 亏 23.7%）
- "热点"开新仓（"你都知道是热点了那必定是中部或高位了"）
- 听别人推荐的（"不听别人说的股票"）
- 高位高换手（"换手率十几" → 故意做出来的，急拉就要走）
- 缩量上涨（"想要涨先有量""量为价先"）

## 4. 仓位、买卖、加减仓规则

### 总仓位结构（按大盘级别）
见上方"大盘判级"表格。

### 加仓（正三角形）
- 中线票升级：观察仓 20% → 突破确认 40% → 回踩不破 60%
- 任何加仓**前提**：不破结构 + 当前盈利

### A 级盘短线 30–40% 仓位的实操路径（数学闭合）

A 级盘"短线 30–40%"是**累积上限**而非"一晚建满"。13.5 万账户 + 单票 ≤ 8% + 单板块 ≤ 25% 的约束下，实际路径：

- **首只**：试错仓 5–6%（约 7000–8000 元 / 1 只）；等当日尾盘看是否守住
- **加第 2 只**（首只浮盈 ≥ 3% 才考虑）：另一个板块的龙头 5–6%
- **加第 3 只**（前两只仍在持有 + 大盘未转弱）：再 5–6%
- **加第 4 只**（同上）：5–6%
- 累计 ~24% 时若集中在 1 个板块 → 触及单板块 25% 上限，强制分散到第 3 个板块；分到 3+ 板块可加到 30–40% 上限

**关键约束**：
- 任何加仓前提：**已持仓有浮盈** + 结构没破
- 用户经验"一定要等持仓赚到钱才可以开新仓"= 这条路径的硬规则
- 大盘当日转 B/C 级 → 停止加仓，已建仓位按结构止损
- 单晚最多新建 2 只票（盘后预案），其余等次日盘中确认
- **反面**：看见 "30–40%" 数字就一晚追 5 只票满仓 → 完全违背"分仓"和"等持仓赚到钱"两条纪律

### 减仓（倒三角形 / 三段式卖法）
针对**有利润**的持仓：
- 股价上涨 5% → 卖 10%
- 股价上涨 10% → 卖 20%
- 股价上涨 15% → 卖 30%

针对**主升浪**的标准化卖法：
1. 强一致日（高开 / 缩量加速 / 群里开始教新人）→ 卖 30%
2. 第一次放量震荡 → 再卖 30%
3. 跌破 5 日线 → 全清

> "**我不赚最后一个铜板，我赚最容易赚的钱。**"卖飞是成本，不是错误。

### 止损 / 止盈
- 短线：**结构止损 + 3% 底线（取严者）** — 结构止损 = 跌破当日 5 日线 / 当日前低 / 分时均线无承接；3% 仅作底线防御（A 股 T+1 + 日内振幅 5–7% 环境下，固定 3% 易被反复触发出局）
- 中线：跌破 60 日线 → 减仓；放量长阴 → 必须处理
- "急拉先卖一部分，后面肯定有调整"
- 控制回撤最保守方案：第一个**放量大阳**就**减仓一半**

### 补仓
- **慢慢补，差距拉开才补**（"几分就出手，子弹全没有"）
- "高位买的少，低位买的多"（三角建仓）
- 上午补的仓**下午拉高要及时出货**（"水满则溢、月满则亏"）
- ❌ **绝对不做倒 T**（先卖后买） — 电网设备 ETF 卖飞案例
- ✅ 被套优先做**正 T 解套**（先买后卖，前提是震荡向上 + 换手率 3–5%）

### 卖飞复进（卖出后股价继续涨怎么办）

A 股日内 / 隔日反复极常见，"卖出后又涨上去"是高频场景。规则：

- **卖出后继续涨** → **一律不追**；承认卖飞成本，不破坏纪律
  - 用户语录："卖飞是成本，不是错误"
  - 反面：追入后再调整 = 既卖飞又站岗，双重亏损
- **卖出后回踩** → 可重新评估介入：
  - 回踩到 5 日线 / 10 日线 / 当日卖出价 -3% → 可介入
  - 复进仓位上限：**原卖出仓位的 50%**（试错性，不押回原仓位）
  - 必须满足原入选所有条件（板块没变、票型没变、结构没破）
- **卖出后破前低** → **永不复进**；结构已坏，认输
- **当日复进**特例：上午补的仓下午拉高 → 当日必须给个处理（不长持过夜）

### 不能做的事（红线）
1. 满仓猛干 — 任何时候都要留预备队
2. 频繁换股 — "稳定一到两只长波段运维"，盘整期警惕见异思迁
3. 畏低少补、追高加仓
4. 根据当日热点开新仓
5. 常规行情放量还不跑
6. 连续 2 笔亏损后还交易 — 强制停手 1 天
7. 盘中临时改变交易计划（本来低吸临时去追高）— 这是最致命的亏损来源
8. 退潮期接力（二板/三板）

## 5. 持仓评估清单（每只过 6 项）

每晚对 holdings.md 中每只持仓走以下 6 项检查：

### A. 趋势是否破坏
- 看 60 日线、20 日线、关键平台支撑
- 跌破 60 日线 + 量放大 → **退回观察仓或全清**
- 跌破 20 日线但 60 日线还在 → 减半观察
- 还在 20 日线上 + 缩量回调 → 持有

### B. 是否触发三段式卖点
- 强一致日（高开 / 缩量加速 / 板块讲故事）→ 卖 30%
- 第一次放量震荡 → 再卖 30%
- 跌破 5 日线 → 全清

### C. 公告风险与催化
- 近 30 天公告里出现"减持/质押/问询/异动/诉讼/担保/业绩预亏" → **降级或减仓**
- 近 30 天有"中标/回购/业绩预增/股权激励" → 加分但不重仓加仓

### D. 板块位置
- 还在主线 → 持有
- 已变成补涨 / 跟风 → 减仓
- 板块整体走弱 + 个股不强 → 出局

### E. 是否适合做 T 解套
适合反复做 T 的特征：
- 小盘（市值 ≤ 100 亿，振幅 3–4 点）
- 换手率 3–5%（活跃但不疯）
- 上升通道（震荡向上）— 下降通道不做 T

> **绝不做倒 T**。被套先做正 T 解套（先买后卖），有利润可以先卖、回落再买，不回落就拉倒。

### F. 仓位再平衡
- 单票总仓 > 15% → 评估是否减仓
- 同板块合计 > 25% → 必须减仓
- 中线对冲仓被动减少（短线赚得多了）→ 检查是否需要补回中线

### G. MACD 顶背驰预警（缠论 ② — 事前卖出信号，2026-06 加）

用 `./scripts/data/kline.sh <code> 60 1d | jq '.data.macd_divergence, .data.chanlun_levels'`：

- **`macd_divergence.top_divergence = true`** → 出现一类卖点信号（价格创新高但 MACD 柱体未跟上）
- **处置**：
  - 减仓 30%（**不全清**，等价格确认）
  - 跌破 5 日线 → 清剩余
- **与三段式卖法的关系**：MACD 顶背驰通常**早于"强一致日"1-3 天**出现，是更前置的减仓信号
- **盲区**：MACD 背驰在震荡市误判率高（峰谷频繁切换），所以必须结合趋势 — 趋势完整 + 顶背驰 = 减仓；震荡市 + 顶背驰 = 不动作
- **如何区分趋势完整 vs 震荡市**：看 `.data.zhongshu.valid`：
  - valid=true（盘整中枢）→ 震荡市，MACD 背驰**仅作参考**
  - valid=false（趋势段）→ 趋势完整，MACD 背驰**当作减仓信号执行**

### 输出格式
对每只持仓输出：
```
代码 名称
- 趋势：[未破 / 跌破 20 日线 / 跌破 60 日线]
- 三段式信号：[未触发 / 强一致日 / 放量震荡 / 跌破 5 日线]
- 公告：[无变化 / 出现 X 风险 / 出现 Y 利好]
- 板块：[主线 / 分支 / 补涨 / 走弱]
- 做 T 适配：[是 / 否（原因）]
- 仓位：[当前 X% / 目标 Y%]
- 建议：[继续持有 / 减仓 X% / 做 T / 止损出局 / 暂不动观察]
- 触发条件：明日盘中出现 ___ 立即 ___
```

## 6. 用户的关键纪律语录（务必尊重）

报告中出现以下情况时，**必须**显式引用对应纪律：

- 出现追高诱惑 → "**踏空就算了，不是模式内的票。**"
- 出现重仓诱惑 → "**乱世之下切记有所保留。**"
- 连扳票诱惑 → "**控制回撤真的很重要。再也不追连扳了。**"
- 大涨想加 → "**水满则溢，月满则亏。**"
- 反弹想追 → "**行情反弹但切勿追涨。**"
- 缩量启动几天 → "**这种缩量启动了几天的股以后不要做了，只做不断放量上涨的。**"
- 大盘指数突破 → "**大盘指数连续上涨突破关键位时，切勿过度兴奋手痒，频繁开仓。心如止水才能花开富贵。**"
- 想短线 → "**最好不要做短线，非要做短线也必须有自己的纪律（设置止损等）。**"
- 想抄底 → "**退潮当天不要抄底，不要在瀑布下洗澡。**"
- 看到一字涨停 → "**应该挂跌停价跑路**（指卖出的时候不要限价吃满利润）。"
- 高位放量 → "**高位放量滞涨是典型的主力出货信号。**"
- 高位低换手 → "可以多拿"；**高位高换手** → "赶紧跑"
- **低位低换** → "不做"；**低位高换** → "可以挖掘"
- 缩量下跌 → "等反弹"；**加量下跌** → "撤军回退"
