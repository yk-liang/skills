# 操作手册（用户经验沉淀的可执行版本）

这份文档把用户 Notion 里的复盘经验拆成可直接执行的判定。每条规则都标注**为什么**——不只为了照办，也为了 agent 在边界场景能合理判断。

---

## 1. 大盘判级（黄白线五眼框架）

### 数据来源
东方财富上证分时图 + 资金流。打开 `https://quote.eastmoney.com/zs000001.html` 看分时图的两条线：

- **白线** = 加权指数（受银行/保险/石油/白酒等权重股影响大）
- **黄线** = 平均股价线（不加权，反映多数个股的真实赚钱效应）

### 五眼检查
1. **黄白线相对位置**：黄≥白（情绪好）、黄<白（赚指数不赚钱）。
2. **10:30 前结构**：高开后横住 = 资金不急跑；拉一下就被砸 = 护指数出货。
3. **成交量**：放量但黄线不涨 = 权重在出货 / 缩量但黄线稳 = 情绪未崩。
4. **资金（BBD/主力净流入）**：净流入 = 真涨；持续负流入但指数稳 = 国家队/权重在扛。
5. **13:30 后是否有回流**：午后量没跟上 = 直接放弃，午后量起来 = 可加仓。

### 定级
| 级别 | 黄白线 | 量 | 资金 | 仓位 | 短线策略 | 中线策略 |
|---|---|---|---|---|---|---|
| **A** | 黄 ≥ 白，共振向上 | 放量 | 净流入或回流 | 短 30–40% / 中 40–50% / 现 10–20% | 可追强、打板、吃情绪 | 可加仓、持股 |
| **B** | 白稳黄略弱 | 一般 | 小幅流出 | 短 10–20% / 中 40% / 现 40–50% | 只做最强分支 / 龙头 | 不加仓但可持有 |
| **C** | 黄明显弱于白 | 缩量或放量下跌 | 持续流出 | 短 0–5% / 中 ≤ 20% / 现 ≥ 70% | 基本不做 | 减仓、等右侧 |

**保命线**（必须执行，不准妥协）：
> **黄线连续 2 天弱于白线 + 量不放 → 所有短线仓位自动清零。**

由于本 skill 是夜间运行，用户已经知道当天黄白线全貌，所以判级看**今日**和**昨日**两天的形态。

## 2. 情绪温度 + 退潮期识别

### 情绪温度指标（每晚必抓）
- 涨停家数（剔除 ST）、跌停家数
- 连板高度（最高板 + 连板≥3 的票数）
- 昨日龙头今日表现（涨停 / 高开不封 / 低开破板）
- 首板晋级率：昨日首板今日二板的比例

### 退潮期判定（命中 ≥ 2 条 → 进入退潮模式）
1. 涨停家数连续下降
2. 昨日龙头**高开不封 / 冲高回落**
3. 昨强板块**无一字、无换手二板**
4. 黄线持续弱于白线
5. 首板晋级率 < 30%

### 退潮期纪律（铁律，不准违反）
- ❌ **不做首板**
- ❌ **不追反包**
- ❌ **不做新题材**
- 退潮期只做 3 类票（按优先级）：
  1. **辨识度最高的老龙反抽**（前 5 板以上 + 跌 30–50% + 首次放量止跌）
  2. **指数对冲型**（银行/石油/电力/中字头/中特估 + 低波动）
  3. **极端情绪错杀**（无明显利空 + 1–2 天暴跌 15%+ + 尾盘有承接）
- 退潮期仓位公式：`涨停家数 × 1% = 当日最大短线仓位`
  - 例：涨停 30 家 → 短线 ≤ 30%；涨停 15 家 → 短线 ≤ 15%

### 情绪冰点的 5 个**提前**信号（命中 ≥ 2 → 立即降仓）
1. 龙头"给你走"了（低开能让你走、冲高不封不杀）—— **最危险的信号**，市场不抢了
2. 二板开始大量断板（首板还行但二板接不上）
3. 强股缩量阴跌（不暴跌但每天跌一点 + 量越来越小）
4. 跌停不多但涨停更少（看似没事，实际没人赚钱）
5. 同题材内卷（无唯一核心，互相卡位）

命中 ≥ 2 → 报告里**显式提醒用户**：
- 短线仓位直接减半
- 不再做二板以上
- 交易周期压缩到隔日

> **冰点 ≠ 抄底点。** 冰点 = 生存考验。真正的机会在冰点之后第一个"能连板的东西"出现时。

## 3. 票型分类与识别

候选股**必须明确归类**到下面某种，不同票型用不同评分维度（详见 scoring.md），并有不同仓位上限。

> **先板块再票型**：每个候选必须先回答"该股属于哪个板块、板块当前阶段是什么"。板块判定见 `references/sector-screening.md`。
> 板块阶段 → 允许票型映射：
> - 主升板块 → 情绪龙头 / 均线启动
> - 新启动板块 → 情绪龙头（A/B 级盘）/ 老龙反抽
> - 修复板块 → 老龙反抽
> - 补涨板块 → 错杀反弹（仓位极低）
> - 轮动 / 退潮板块 → 不选个股
> - 中线慢牛 / 护盘衍生 / 逆市不跌不绑定板块阶段（独立筛选逻辑）

### 票型 1：情绪龙头
- 触发：所在板块涨停家数第一 + 分时第一次分歧 + 不看估值只看承接
- **限定 A 级或强 B 级盘**，C 级直接跳过
- 单票仓位 ≤ 8%，止损 ≤ 3%

### 票型 2：老龙反抽
- 触发：前期 5 板以上 + 跌幅 30–50% + 某天首次放量止跌 + 黄线不再创新低
- **退潮期首选**，但只做隔日 / 两日，反抽 5–10% 就走
- 单票仓位 ≤ 10%

### 票型 3：错杀反弹
- 触发：无明显利空 + 1–2 天暴跌 15%+ + 尾盘有承接
- 只做隔日，止损极严（−3%）
- 单票仓位 ≤ 5%

### 票型 4：护盘衍生 / 中特估对冲
- 触发：C 级或退潮期 + 银行/石油/电力/中字头/中特估 + 低波动（日振幅 ≤ 3%）
- 当避风港用，不格局
- 单票仓位 ≤ 15%

### 票型 5：中线慢牛（账户的"压舱石"）
- 触发条件（必须**全部**满足）：
  1. 周线没走坏
  2. 日线在 20 / 60 日线上
  3. 回调缩量、上涨放量
  4. 财报：营收 + 归母 + 扣非同比双正，经营现金流为正
  5. 市场共识高（机构票，不纯炒作）
  6. 日振幅 ≤ 3%（低波动）
- **观察仓 → 主仓**升级条件（满足下列 5 选 4）：
  1. 至少一个指数站回 60 日线 + 3 天不破
  2. 黄白线出现至少 1 天共振
  3. 回调缩量 + 突破放量（不是一日游）
  4. 连续 3 天涨多跌少 + 赚钱效应 > 50%
  5. 同一主线持续 5–7 天（非一日题材）
- 仓位节奏：
  - 观察仓 20% → 突破确认加到 40% → 回踩不破加到 60%
- 退仓条件（任一触发）：
  1. 跌破 60 日线
  2. 放量长阴
  3. 指数转 C 级 → 主仓退回观察仓 20%

### 票型 6：均线启动（短线模式）
- 触发：5 日线上穿 10 日线 + 板块共振 + 第二天板块继续走强
- **要等"成功穿上且走稳"**——刚穿上去经常马上又下来
- 信号最强：所有均线（5/10/20/30/60）多头排列 + 股价在均线上 → 一直拿，跌破 10 日线再走
- 均线**低位缠绕**还好（向上概率大），**高位缠绕**是分歧（不要做）

### 票型 7：逆市不跌（缠论思路）
- 触发：大盘调整时这只票横盘不跌、量能温和、无负面公告
- 思路：下轮黑马由此产生概率高（参考永鼎案例 — 3 月行情差时它能横盘不跌就是强）
- 进入观察池，**不主动追**，等大盘企稳后第一个放量大阳跟进
- 单票仓位 ≤ 8%

### 不允许选的（绝对禁区）
- 缩量启动几天后才放量介入的（新能泰山案例 — 一天涨停一天跌停，亏 20%）
- 连扳后追高（东方明珠案例 — 亏 23.7%）
- "热点"开新仓（"你都知道是热点了那必定是中部或高位了"）
- 听别人推荐的（"不听别人说的股票"）
- 高位高换手（"换手率十几" → 故意做出来的，急拉就要走）
- 缩量上涨（"想要涨先有量""量为价先"）

## 4. 仓位、买卖、加减仓规则

### 总仓位结构（按大盘级别）
见上方"大盘判级"表格。

### 加仓（正三角形）
- 中线票升级：观察仓 20% → 突破确认 40% → 回踩不破 60%
- 任何加仓**前提**：不破结构 + 当前盈利

### 减仓（倒三角形 / 三段式卖法）
针对**有利润**的持仓：
- 股价上涨 5% → 卖 10%
- 股价上涨 10% → 卖 20%
- 股价上涨 15% → 卖 30%

针对**主升浪**的标准化卖法：
1. 强一致日（高开 / 缩量加速 / 群里开始教新人）→ 卖 30%
2. 第一次放量震荡 → 再卖 30%
3. 跌破 5 日线 → 全清

> "**我不赚最后一个铜板，我赚最容易赚的钱。**"卖飞是成本，不是错误。

### 止损 / 止盈
- 短线初始止损 ≤ 3%（违反必亏）
- 中线：跌破 60 日线 → 减仓；放量长阴 → 必须处理
- "急拉先卖一部分，后面肯定有调整"
- 控制回撤最保守方案：第一个**放量大阳**就**减仓一半**

### 补仓
- **慢慢补，差距拉开才补**（"几分就出手，子弹全没有"）
- "高位买的少，低位买的多"（三角建仓）
- 上午补的仓**下午拉高要及时出货**（"水满则溢、月满则亏"）
- ❌ **绝对不做倒 T**（先卖后买） — 电网设备 ETF 卖飞案例
- ✅ 被套优先做**正 T 解套**（先买后卖，前提是震荡向上 + 换手率 3–5%）

### 不能做的事（红线）
1. 满仓猛干 — 任何时候都要留预备队
2. 频繁换股 — "稳定一到两只长波段运维"，盘整期警惕见异思迁
3. 畏低少补、追高加仓
4. 根据当日热点开新仓
5. 常规行情放量还不跑
6. 连续 2 笔亏损后还交易 — 强制停手 1 天
7. 盘中临时改变交易计划（本来低吸临时去追高）— 这是最致命的亏损来源
8. 退潮期接力（二板/三板）

## 5. 持仓评估清单（每只过 6 项）

每晚对 holdings.md 中每只持仓走以下 6 项检查：

### A. 趋势是否破坏
- 看 60 日线、20 日线、关键平台支撑
- 跌破 60 日线 + 量放大 → **退回观察仓或全清**
- 跌破 20 日线但 60 日线还在 → 减半观察
- 还在 20 日线上 + 缩量回调 → 持有

### B. 是否触发三段式卖点
- 强一致日（高开 / 缩量加速 / 板块讲故事）→ 卖 30%
- 第一次放量震荡 → 再卖 30%
- 跌破 5 日线 → 全清

### C. 公告风险与催化
- 近 30 天公告里出现"减持/质押/问询/异动/诉讼/担保/业绩预亏" → **降级或减仓**
- 近 30 天有"中标/回购/业绩预增/股权激励" → 加分但不重仓加仓

### D. 板块位置
- 还在主线 → 持有
- 已变成补涨 / 跟风 → 减仓
- 板块整体走弱 + 个股不强 → 出局

### E. 是否适合做 T 解套
适合反复做 T 的特征：
- 小盘（市值 ≤ 100 亿，振幅 3–4 点）
- 换手率 3–5%（活跃但不疯）
- 上升通道（震荡向上）— 下降通道不做 T

> **绝不做倒 T**。被套先做正 T 解套（先买后卖），有利润可以先卖、回落再买，不回落就拉倒。

### F. 仓位再平衡
- 单票总仓 > 15% → 评估是否减仓
- 同板块合计 > 25% → 必须减仓
- 中线对冲仓被动减少（短线赚得多了）→ 检查是否需要补回中线

### 输出格式
对每只持仓输出：
```
代码 名称
- 趋势：[未破 / 跌破 20 日线 / 跌破 60 日线]
- 三段式信号：[未触发 / 强一致日 / 放量震荡 / 跌破 5 日线]
- 公告：[无变化 / 出现 X 风险 / 出现 Y 利好]
- 板块：[主线 / 分支 / 补涨 / 走弱]
- 做 T 适配：[是 / 否（原因）]
- 仓位：[当前 X% / 目标 Y%]
- 建议：[继续持有 / 减仓 X% / 做 T / 止损出局 / 暂不动观察]
- 触发条件：明日盘中出现 ___ 立即 ___
```

## 6. 用户的关键纪律语录（务必尊重）

报告中出现以下情况时，**必须**显式引用对应纪律：

- 出现追高诱惑 → "**踏空就算了，不是模式内的票。**"
- 出现重仓诱惑 → "**乱世之下切记有所保留。**"
- 连扳票诱惑 → "**控制回撤真的很重要。再也不追连扳了。**"
- 大涨想加 → "**水满则溢，月满则亏。**"
- 反弹想追 → "**行情反弹但切勿追涨。**"
- 缩量启动几天 → "**这种缩量启动了几天的股以后不要做了，只做不断放量上涨的。**"
- 大盘指数突破 → "**大盘指数连续上涨突破关键位时，切勿过度兴奋手痒，频繁开仓。心如止水才能花开富贵。**"
- 想短线 → "**最好不要做短线，非要做短线也必须有自己的纪律（设置止损等）。**"
- 想抄底 → "**退潮当天不要抄底，不要在瀑布下洗澡。**"
- 看到一字涨停 → "**应该挂跌停价跑路**（指卖出的时候不要限价吃满利润）。"
- 高位放量 → "**高位放量滞涨是典型的主力出货信号。**"
- 高位低换手 → "可以多拿"；**高位高换手** → "赶紧跑"
- **低位低换** → "不做"；**低位高换** → "可以挖掘"
- 缩量下跌 → "等反弹"；**加量下跌** → "撤军回退"
