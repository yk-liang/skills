# 用户历史案例库（模式识别参考）

每条案例都对应一种**模式**，agent 在选股时若发现新候选符合"失败模式"应**直接回避**，符合"成功模式"才放心入选。

新案例应追加到对应模式之下，便于积累。日期保留用户 Notion 原始日期。

---

## 失败模式

### 案例 1：缩量启动几天后才放量介入 → 一天涨停一天跌停 → 亏 20%
- **个股**：新能泰山（2026.3.31）
- **错误**：缩量一字启动了几天后，在放量当天介入
- **后果**：精准踏错节奏，亏 20%
- **沉淀规则**：**只做不断放量上涨的票**。缩量启动几天的，启动期没买就放弃，放量再追是 99% 接盘
- **agent 落地**：在东财涨停板池（zt_pool）取近 5 日涨停股 → 对每只用东财个股页 / 同花顺个股页拉日 K 数据，比较 5 日均量 vs 20 日均量；若 5 日均量 < 20 日均量 + 仍处涨停趋势 → 标红进回避池

### 案例 2：追连扳 → 控不住回撤
- **个股**：东方明珠（2026.1.21）
- **错误**：追连扳，连扳后追高
- **后果**：亏 23.7%，至复盘当日仍在跌
- **沉淀规则**：**再也不追连扳**。等企稳后逢低吸收，逢高出掉
- **agent 落地**：candidate 出现"近 5 日 ≥ 3 个涨停且最新一日仍涨停" → 直接进回避池

### 案例 3：早盘冲动追进 → 多亏一百多
- **个股**：（2026.3.26 涨停板）
- **错误**：本来挂了 5 日线低吸单，看到开盘拉升按捺不住直接追进去
- **沉淀规则**：**忍住，等回踩关键位置看承接情况。踏空就算了，不是模式内的票**
- **agent 落地**：报告里给的"试错仓位"必须明确"等回踩 X 日线再分批介入"，不准写"开盘冲进"

### 案例 4：贪婪吃满 → 卖飞收益归零
- **个股**：量子科技（2026.3.20 一字板）
- **错误**：竞价撤单后挂涨停价卖（幻想回封），结果一路下砸
- **沉淀规则**：**一字板想卖就挂跌停价跑路，不要限价吃满。先保收益**
- **agent 落地**：持仓评估时，如果一字涨停 + 集合竞价封板 < 50% → 建议"挂跌停价果断出"

### 案例 5：太想"一蹴而就"扭亏 → 不减仓继续亏
- **个股**：福莱特（2026.1.30）
- **错误**：想直接扭亏为盈，不减仓
- **沉淀规则**：**步步为营不断降成本。大幅拉升先减仓，不要贪**
- **agent 落地**：持仓还在亏损但当日大涨 ≥ 5%，建议"减一半，回落再补"

### 案例 6：诱多套人
- **个股**：浪潮（2026.3.17）
- **错误**：以为中午收盘前的拉升是真的
- **沉淀规则**：**中午收盘前的拉升要警惕诱多套人**
- **agent 落地**：尾盘出现高位放量但收盘附近回落 → 标"诱多疑似"

### 案例 7：同样错误重复出现
- **个股**：诺德股份（2026.4.8 + 2026.4.21 两次）
- **错误**：趋势没走坏却频繁动手
- **沉淀规则**：**趋势没走坏真的不要着急乱动，有盈利更是可以等盘尾再看情况**
- **agent 落地**：持仓评估"做 T"建议时，如果当日股价仍在 5/10/20 日线之上 → 默认建议"盘尾再决定"

### 案例 8：分时震荡破坏模式
- **个股**：深圳能源（2026.3.27）
- **错误**：快速上冲回落、震荡回踩均线却没有接着上冲，已不符合模式
- **沉淀规则**：**当模式被破坏时该出就出**。上午冲到 +7% 时其实可以分批减 1/5
- **agent 落地**：持仓评估时，"早盘 +5% 但盘中回吐到分时均线附近无承接" → 建议"减 1/5"

## 成功模式

### 案例 9：横盘不跌 = 强势信号
- **个股**：永鼎股份（2026.3.26 关注）
- **观察**：3.26 之前行情一直不好，但永鼎能横盘不跌 → 本身就代表它强
- **沉淀规则**：**大盘调整时密切关注逆市不跌的股票，下轮黑马由此产生**
- **agent 落地**：先看上证 / 深证近 10 日是否下跌；如果大盘下跌，从东财行业板块今日**跌幅小于 -1%**的板块里取成分股，进个股页看：5/10/20 日 K 线是否横盘（振幅 ≤ 5%）+ 量持续萎缩 + 主板代码 → 进"逆市不跌"观察池

### 案例 10：反包大阴线 = 抛压衰竭
- **沉淀规则**：**上升趋势中出现"大成交量的大阴线"被缩量反包**，意味着大阴线那天交换筹码进去的资金没卖、抛压已衰竭
- **下跌趋势谨慎**：下跌趋势的反包未必好，除非是打到关键支撑出现缩量反包
- **agent 落地**：从东财今日涨幅榜（个股榜）筛主板 + 阳线反包 + 收盘高于昨日开盘 → 逐个进个股页验证：昨日大阴线 + 量大、今日阳线 + 量比昨日小、收盘价站在 60 日均线上 → 加分项

### 案例 11：5/10 均线金叉 + 板块共振
- **沉淀规则**：5 日线上穿 10 日线是进场信号，但要"成功穿上且走稳"+ 板块第二天继续走强
- **信号最强**：所有均线多头排列 + 股价在均线上 → 一直拿，跌破 10 日线再走
- **agent 落地**："均线启动"票型的入选条件就是这条

## 模式归纳：什么时候该买、什么时候该卖

| 信号 | 含义 |
|---|---|
| 跌停**放量** | 分歧得到消化，可观察反弹 |
| 跌停**缩量** | 还要跌，分歧没消化完 |
| 上升趋势 + 大阴线被缩量反包 | 抛压衰竭，可观察 |
| 高位**放量滞涨** | 出货信号，必须减仓 |
| 高位放量但封板 / 大阳 → 缩量回调 | 还有空间，可把握 |
| 低位出利好 | 问题不大 |
| **高位出利好** | 出货嫌疑 |
| 阴转阳第一天 | 不必急着追，等量起来 |
| 缩量下跌 | 等反弹 |
| 加量下跌（量越大跌得越狠） | 撤军回退时 |
| 急拉 | 卖出一部分，后面肯定有调整 |
| 断板**没放量** | 温和断板，可能走成趋势 |
| 断板**放大量** | 派发完成，结束 |
| 第一个**放量大阳** | 控制回撤的减仓信号（保守方案：减一半） |
