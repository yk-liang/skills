# 板块筛选（先板块，再个股）

> "**和板块趋势相悖是不明智的。**" — 大作手
>
> "**不要去买那些没有追随板块领涨股的股票。**" — 用户经验

板块定位是选个股**之前**的先决步骤。板块不强 → 个股不选；板块走弱 → 持仓减仓。这一步做错，后面所有评分都是无用功。

## 板块筛选 6 步流程

### Step 1 — 拉今日涨幅榜
打开**概念板块涨幅榜**和**行业板块涨幅榜**，分别取：
- 涨幅 Top 10
- 跌幅 Top 5（看哪些行业在被抛弃）
- 成交额 Top 10（资金流向哪里）

输出：当日涨幅前 10、跌幅前 5、放量前 10（按这三个维度交叉看）。

### Step 2 — 给每个候选板块定"阶段"
对涨幅 Top 10 + 成交额 Top 10 的并集（去重后约 10–15 个板块）逐个分类：

| 阶段 | 判定特征 | 处置 |
|---|---|---|
| **主升（Mainline）** | 连续 3 天以上领涨 + 涨停家数 ≥ 3 + 龙头明确 + 持续放量 | **可重仓**，选龙头或核心分支 |
| **新启动（Initiator）** | 今天首次脱颖而出（昨天还不在涨幅榜） + 龙头连板 ≥ 2 + 概念有公开催化（政策/事件/数据） | 谨慎试错（≤ 5–8% 仓位），等明日确认 |
| **补涨（Laggard）** | 主升板块已走 3 天 + 该板块今天才动 + 板块内补涨股温和涨幅（5–7%） | **慎入**，只适合做隔日 / 两日 |
| **轮动（Rotation）** | 一日游、当日强但无连板、明显跟风风格 | **不做** |
| **退潮（Ebbing）** | 前期主线但今日明显走弱 + 龙头炸板 + 板内 ≥ 2 只跌停 | **持仓必须减仓**，新仓不进 |
| **修复（Recovery）** | 已超跌 30%+ + 今日首次放量止跌 + 龙头日内未破前低 | 适合做老龙反抽（仅退潮期） |

**判定数据怎么拿**：
- 板块"涨停家数" → 涨停板池按概念板块统计
- 板块"龙头连板" → 涨停板池里按板块过滤后看连板数最高的
- 板块"是否新启动" → 看东财板块详情页的近 5 日 K 线（板块自己也有日 K）
- 板块"持续天数" → 看近 5 日板块涨幅，连续 3 天涨 ≥ 1.5% 算持续

### Step 2.5 — 缠论中枢化客观判定（2026-06 加，推荐替代 Step 2 主观判定）

主观词容易在 agent 之间产生分歧（同一板块"主升"vs"新启动"判断不同）。用 `sector_kline.sh` 输出的 `zhongshu` + `chanlun_levels` 字段做**客观判定**：

```bash
./scripts/data/sector_kline.sh BK1136 60 | jq '{
  macd_trend: .data.macd_last.trend,
  top_div: .data.macd_divergence.top_divergence,
  bottom_div: .data.macd_divergence.bottom_divergence,
  zhongshu: .data.zhongshu,
  chanlun: .data.chanlun_levels
}'
```

**输出 → 阶段映射**：

| `chanlun_levels` / `macd_last` / `zhongshu` 状态 | 客观对应阶段 |
|---|---|
| 三类买点 + macd 金叉 / 多头 | **主升 / 新启动** |
| 一类买点（底背驰）+ 当前价在中枢下沿附近 | **修复** |
| 中枢 valid=true + chanlun_levels 为空 + 量缩 | **轮动** |
| 三类卖点 / 收盘价 < 中枢下沿 | **退潮** |
| 一类卖点（顶背驰）+ 价格在中枢上沿附近 | **主升末端**（持仓减仓） |
| 突破中枢上沿但未回踩 | **新启动**（仓位减半试错） |

**主观判定（Step 2）与客观判定（Step 2.5）冲突时**：
- 以**客观为准**写入报告主结论
- agent 在报告里**两套都列**（"主观判：X / 客观判：Y"），让用户复核
- 出现冲突的板块标"低置信度"

注：板块阶段保留 6 阶段定义（主升/新启动/补涨/轮动/退潮/修复），改的只是**判定方法**（从"看 K 线感觉"到"看脚本输出"）。

### Step 3 — 检查板块**宽度**（breadth）
对每个候选板块，打开成分股列表，看：
- 板内总股数
- 板内**今日上涨家数 / 总数**比例（宽度指标）
- 板内**今日涨停家数**

| 宽度 | 含义 | 处置 |
|---|---|---|
| > 70% 上涨 + ≥ 3 个涨停 | 真主升，板块共振 | 可在板内选个股 |
| 50–70% 上涨 + 1–2 个涨停 | 板块有动能但分化 | 只选最强 1–2 只 |
| < 50% 上涨 即使板块涨幅大 | 是个别股拉的板块涨幅 | **不入选**（伪板块） |

### Step 4 — 检查板块**持续性**
看板块近 5 个交易日 K 线：
- 连续 3 天以上放量上涨 → 持续性强
- 上涨但量逐日萎缩 → 接近末端
- 一日游 → 不参与

### Step 5 — 北向 / 龙虎榜验证
- 北向资金今日净买入行业偏好里有该板块 → 加分
- 龙虎榜上同板块多只票被知名游资席位 / 机构席位买入 → 加分
- 龙虎榜上**机构净卖出**该板块多只 → 警惕，板块可能进入派发期

### Step 6 — 输出板块结论
报告里写一段板块概览（约 5–10 行）：

```
## 板块概览

主升：核电（已持续 4 天，板内 24/35 上涨，连板梯队最高 3 板，龙头 X / Y）
     军工（持续 3 天，板内 18/40 上涨，连板 2 板）
新启动：固态电池（首日，龙头 X 涨停，催化 = 工信部 X 月文件）
补涨：光伏（昨日核电的补涨方向，板内 15/30 上涨，温和涨幅）
退潮：AI 算力（板块今日跌幅 -2.3%，前期龙头 X 炸板）
修复：（无）
```

## 板块 → 个股的映射规则

确定板块阶段后，从板块**内部**选个股，不从全市场扫描：

| 板块阶段 | 在板内选什么票型 |
|---|---|
| 主升 | **短线龙头**（子型 A：板内连板第一龙头；子型 B：板内 5/10 金叉个股） |
| 新启动 | **短线龙头子型 A**（仅 A 级盘 + 板内涨停家数第一 + 板块当日新启动）+ **老龙反抽子型 A**（如该板块前期有 5 板以上龙头） |
| 补涨 | **老龙反抽子型 C**（板内尚未启动的低位主板股）— 仓位减半 ≤ 5% |
| 轮动 | 不选 |
| 退潮 | **不选短线**；护盘衍生独立筛选；中线慢牛若属该板块 → 持仓减仓 |
| 修复 | **老龙反抽子型 A**（板内前期 5 板以上、跌幅 30–50%、首次放量止跌） |

**任何票型都必须先回答："这只票属于哪个板块、板块当前阶段是什么？"** 答不出来 / 板块在轮动或退潮 → 不入选。

## 用户已确认的板块纪律（必守）

1. **板块都在涨它纹丝不动 → 弱势 → 不要幻想后涨**
2. **同板块合计仓位 ≤ 25%**（强制约束，不留弹性）
3. **板块都跌它逆市不跌 → 强势 → 进观察池但不主动追**（缠论，永鼎案例）
4. **板块走 3 天以上才补涨进 → 接最后一棒**，不做
5. **退潮期"不做新题材"** — 即使板块涨幅榜第一也不做（退潮三不做之一）

---

## 失败模式（板块层）

- 看到一个板块涨幅榜第一就冲进去 → 大概率已经是主升中后段
- 板块当日涨 5% 但只有 2 只股拉得，其它 38 只都不动 → 这是**伪板块**
- 板块连续 5 天以上还在涨 → 大概率退潮临近
- "热点开新仓" → 你都看到是热点了那必定是中部或高位（用户经验）

## 成功模式（板块层）

- 板块今日**首次**在涨幅榜露脸 + 龙头封死涨停 + 板内宽度 > 60% + 有公开催化（政策/事件/数据） → 新启动主升候选
- 板块昨天才领涨 + 今天再创新高 + 量没缩 + 龙头继续涨 → 主升第二天，可介入分支
- 大盘下跌中板块逆市不跌（缩量横盘）→ 进观察池等大盘企稳后第一个放量大阳跟进

## 反向信号识别（板块筛选的辅助维度）

除了正向找强势板块，也要识别反向信号 — A 股主板的 alpha 来源之一往往在"逆向"而非"正向"。这三类信号都不直接入选，**进观察池等下次放量确认**（"放量大阳第一天"），介入时归短线龙头管理。

### 信号 1：普跌中的逆向放量
- **触发**：大盘下跌（上证 < -0.5%）+ 单只主板票放量上涨 > 3% + 当日成交量 > 5 日均量 1.5×
- **意义**：全市场风险偏好低时仍敢买 → 资金有特殊确定性
- **数据获取**：`quote.sh` 全市场过滤上涨 + 高成交量

### 信号 2：主升板块里的"非龙头接力"
- **触发**：板块连涨 ≥ 2 天 + 龙头封板 + 板内昨日**未涨停**的票今日放量上涨 > 7%
- **意义**：龙头之外的接力候选，往往是"二线龙头"
- **数据获取**：`sector_constituents.sh` 排序，挑昨日 change_pct < 5% 但今日 > 7% 的票

### 信号 3：板块整体回调中的单票拒绝下跌
- **触发**：板块当日 -2% 以上 + 单只主板票横盘或微涨
- **与信号 1 区别**：信号 1 是"全市场普跌中的反向"，信号 3 是"板块普跌中的反向"
- **意义**：板块内独立强势 → 资金转移到该票（板块结构内部分化）
- **数据获取**：`sector_constituents.sh` 找 change_pct ≥ 0 的票

**共同原则**：反向信号都进观察池 + "逆市不跌"标签，**不主动追**。等到第一个"放量大阳"确认日（大盘配合 + 板块配合 + 个股放量上涨）才介入，介入时按短线龙头票型管理（≤ 8% 仓位 + 结构止损）。
